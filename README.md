# Python Challenges Repository

## Description
This repository contains Python solutions for three coding challenges. The challenges cover various concepts, including time conversion, conditional logic, and string manipulation.

Challenges
Challenge 1: Converting 12-hour time to 24-hour time
This challenge involves converting a 12-hour time format to a 24-hour format. You'll find the solution in the function convert_to_24_hour in challenges.py.

Challenge 2: Two numbers are positive
The second challenge checks if exactly two out of three given numbers are positive. The solution is implemented in the function two_positive in challenges.py.

Challenge 3: Consonant value
For the third challenge, the task is to find the highest value of consonant substrings in a given string. The solution is provided in the function consonant_value in challenges.py.
